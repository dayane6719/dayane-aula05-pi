##  Instalação do Git

O Git foi instalado para permitir o controle de versões do projeto.
Após a instalação, foi possível utilizar comandos como `git clone`, `git add` e `git commit`.

##  Configuração do Git

Após instalar, foi necessário configurar o nome e o e-mail com os comandos:

```bash
git config --global user.name "Seu Nome"
git config --global user.email "seuemail@email.com"
```

Isso é importante para identificar quem fez as alterações no projeto.

##  Criação da conta no GitHub

Foi criada uma conta no GitHub, que é uma plataforma para armazenar repositórios online e facilitar o trabalho em equipe.

##  Criação do repositório

Foi criado um repositório público para armazenar os arquivos do projeto.

##  Clonagem do repositório

O repositório foi clonado para o computador utilizando o comando:

```bash
git clone link-do-repositorio
```

##  Uso do VS Code

O Visual Studio Code foi utilizado como editor de código para criar e editar os arquivos do projeto.

##  Integração com Git

O VS Code foi integrado com o Git, permitindo realizar commits, push e pull diretamente pela interface.

##  Criação de branch

Foi criada uma branch para desenvolvimento, separando as alterações da branch principal.

##  Teste de commits

Foram realizados commits com mensagens descritivas para registrar as alterações feitas.

##  Envio para o GitHub

As alterações foram enviadas para o repositório remoto utilizando o comando:

```bash
git push origin desenvolvimento
```

##  Atualização do projeto

O projeto foi atualizado utilizando:

```bash
git pull
```

##  Conclusão

A configuração do ambiente permitiu preparar o computador para trabalhar com Git e GitHub, facilitando o versionamento e a colaboração no projeto.
